# Conky-Widget
My custom Conky Widget on Ubuntu 18.04 LTS deskop. 

## Install:

```bash
sudo apt update
sudo apt install conky-all
conky

nano ~/.conkyrc #widget appearance customization
```
For more information, read [**here**](https://itsfoss.com/conky-gui-ubuntu-1304/)
